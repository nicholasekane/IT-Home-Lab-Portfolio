# IT Support Home Lab Portfolio

---

## Overview

This repository contains a structured IT support home lab built to practice, document, and demonstrate common service desk, desktop support, and entry-level IT support workflows.

The primary goal of this project is to build legitimate hands-on experience with common enterprise tools in a controlled, non-production environment while creating recruiter-friendly proof of work.

This lab focuses on:

- Active Directory administration
- Windows client support
- Account troubleshooting
- Identity and access management concepts
- Ticket documentation workflows
- Endpoint management fundamentals
- Technical documentation

The purpose of this lab is to build practical entry-level IT support skills through direct hands-on practice.
