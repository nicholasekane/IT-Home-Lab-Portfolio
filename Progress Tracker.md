# Lab Type Definitions

## Hands-On
You personally completed tasks inside a live lab environment such as:

- VirtualBox
- Windows Server
- Active Directory
- Windows Client
- ServiceNow

---

## Documentation Lab
Used when live tenant access was unavailable and learning was completed through:

- Official documentation
- Platform walkthroughs
- Research-based learning
- Structured note taking

## Medium

## High

- What was built
- Why it matters
- Troubleshooting performed
